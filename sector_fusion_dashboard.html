方法论与局限 (数据源 · 强度计算 · 个股选择 · 风控标签 · 局限)

板块强度(三层漏斗 + 扩散门槛,463行业III→今日强势板块)

① 硬过滤(一票否决): C>0 ∩ 1.2≤放量倍数≤3.5 ∩ 日涨≤7% ∩ max(C5超额排名,RS相对强度)前30%

② 强度分(0-100): C5排名(25%)+RS排名(25%)+放量排名(25%)+情绪分(25%),四维等权

③ 扩散门槛(新): 板块进主榜须「通过硬过滤的强叶子数≥2 且 占板块总叶子≥10%」(小板块总叶≤8放宽到≥1)。不达标→观察池。
为何加: 旧版板块强度=成员叶子强度均值,导致只过1个超热叶子的板块(如化工/公路货运)靠极值霸榜,稀释了多叶共振的真主线(半导体4叶)。扩散门槛把"靠单点极值冲高"的板块降级,让多叶共振主线自然顶上,也让本看板与「行业预警」讲同一个故事。

④ 风控标签: 末端拥挤 / 加速过热 / 量价背离 / 情绪转弱 / 大盘逆风 → 聚合为板块「可参与性」

个股选择(硬规则 + 相对板块超额序)

① 板块走势锚 = 对应东财行业板块指数近20日日收益序列

② 个股 = 该板块成分股(按量比取活跃TOP18)近20日日收益序列

③ 入选门槛 = Pearson 相关性 ρ(个股,板块)≥0.5(跟随性硬门,剔除完全不跟板块的噪声票)

④ 排序(改): 第一序 = RS20 相对板块超额(个股20日收益−板块20日收益,板块内择强最有信息量的量);第二序 = 相关性ρ(同等超额取更稳的跟随者)。
为何改: 旧版用纯相关/混合z排序,相关高≠会涨,且对极端值敏感、易选到最拥挤的补涨票。改以"跑赢板块多少"为主序,直接回答"板块内买哪只"。

趋势状态判定(标注层 · 专避"低位反弹冒充多头")

对选出的每只龙头抓130根日K,做60日 ln(收盘) 线性回归 → 年化斜率 + R²(趋势一致性),叠加60日区间位置 pos:

主升趋势=斜率≥30%&R²≥0.35&pos≥0.6&RS≥0;趋势向上=斜率≥10%&R²≥0.2&pos≥0.35;低位反弹=斜率向上但pos<0.35(刚离底未站稳,醒目警示);弱势相关=斜率<10%或RS弱;下行风险=斜率≤-10%&R²≥0.2。

注:趋势仅作标注,不改谁被选中(选股门槛=相关性);用均线排列做生死判据太滞后,故改用回归斜率+R²。趋势强≠能买,位置风险见各板块风控标签。

局限

个股按东财行业板块名匹配通达信细分行业,极少数无对应板块会标注;相关性/RS为20日窗口、趋势为60日窗口,均周度重算;个股不含基本面深扒,实盘前请结合基本面与风控标签。
